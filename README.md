## Objetivo
  Implementar un collage arrastrando y soltando imágenes.
  
## **Herramientas Tecnológicas**
  * HTML 5

  * CSS 3

  * Framework Bootstrap

  * Javascript

  * Babel

  * Node.js

  * Firebase

  * Guía de Estilos(`.eslintrc`)
